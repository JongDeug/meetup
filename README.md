# 🧑‍💻 모임의 목적은?

이번 모임은 **TypeScript 기반 백엔드 개발자**들의 커리어 성장과 네트워킹을 위한 자리입니다.  
백엔드 위주이긴 하지만, 프론트엔드 및 풀스택 개발자분들의 참여도 환영입니다!  
기술과 경험, 고민과 목표를 함께 나눌 수 있는 오픈 커뮤니티를 만들어 가고자 합니다.
